## Установка

Для установки проекта выполните:

```
make setup
```

## Запуск проекта

```
make up
```

## Инициализация

После того как докер образы развернулись, можно выполнить миграции, сидеры и индексацию Meilisearch с помощью:
```
make init
```

> Примечание: Сидеры по умолчанию настроены на создание 1кк записей для демонстрации работы Meilisearch.
